# Portfolio Đinh Lê Thảo Nguyên

Portfolio môn **Nhập môn Công nghệ số và Ứng dụng AI**.

## Chạy cục bộ

Mở `index.html` trực tiếp hoặc chạy:

```bash
python -m http.server 8000
```

Sau đó truy cập `http://localhost:8000`.

## Deploy GitHub Pages

1. Tạo repository mới và đẩy toàn bộ thư mục này lên nhánh `main`.
2. Vào **Settings → Pages**.
3. Ở mục **Build and deployment**, chọn **GitHub Actions**.
4. Workflow trong `.github/workflows/deploy.yml` sẽ tự động xuất bản website.

Trang web dùng HTML/CSS/JavaScript thuần, không cần cài package hay build.
